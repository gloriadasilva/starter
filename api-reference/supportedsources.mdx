1.

2. The supported external sources for **TV shows** include IMDb, Facebook, Instagram, TheTVDB, TikTok, Twitter, Wikidata, and YouTube.

3. The supported external sources for **movies** include IMDB, Facebook, Instagram, TikTok, Twitter, Wikidata, and YouTube.

4. The supported external sources for **TV** seasons include TheTVDB and Wikidata.

5. The supported external sources for **TV** episodes include IMDb, TheTVDB, and Wikidata.

6. The supported external sources for people include IMDb, Facebook, Instagram, TheTVDB, TikTok, Twitter, Wikidata, and YouTube.
