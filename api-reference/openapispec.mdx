The Openapi specification is a standardized format for describing APIs. It is recognized and accepted by different API platforms.

You can use the openapi specification file of TMDB API on different API platforms, such as Redocly, Readme, Stoplight, etc.

Click [here](https://raw.githubusercontent.com/gloriadasilva/starter/refs/heads/main/api-reference/openapi.yaml) to get started.
